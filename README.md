# PRIME project

# Installation

1. Install node.js (>= 8), npm (or yarn)
2. Install [`truffle`](http://truffleframework.com) globally: `yarn global add truffle`
3. Install packages: `yarn`

# Testing

To run tests you need to run `geth` in test mode: `./scripts/testrpc.sh`, then run `yarn test` ro tun tests
